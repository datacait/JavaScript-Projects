This is a unit converter tool built during the Scrimba Intro to Javascript course. It was designed and built based on the given Figma file (https://www.figma.com/file/cqtGul0V8RFXY4vTcIv1Kc/Unit-Conversion?node-id=0%3A1) and the given parameters: 
1) Generate all conversions for length, volumne, and mass when user clicks "Convert"
2) Round the numbers down to three decimal places.
